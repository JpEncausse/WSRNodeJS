Le module XBMC permet de piloter XBMC avec Sarah; pour un meilleur confort, il est possible d'importer les artistes et les genres pour générer le xbmc.xml

La génération se fait manuellement (IP et Port à modifier!):

  • http://127.0.0.1:8080/sarah/xbmc?action=xml_artist
  • http://127.0.0.1:8080/sarah/xbmc?action=xml_genre

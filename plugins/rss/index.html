Le module RSS permet de maintenir un fichier contenant 20 items RSS. Le script accepte les paramètres title, description, url, guid et date.

 

IFTTT

Si le fichier est mis dans le répertoire public d'une DropBox il sera alors accessible depuis le web (en http pas https) par des services comme IFTTT.

C'est donc un pont entre les objets de la maison et les services web qui complète le module calendrier.

Le module fitbit permet de lire les données FitBit

Pour le moment:

  1. SARAH combien j'ai marché de pas aujourd'hui ?
  2. SARAH combien j'ai monté d'étage aujourd'hui ?

Enregistrement

SARAH ne marche pour le moment qu'avec 1 FitBit. On pourra imaginer par la suite utiliser la recofaciale pour en gérer plusieurs.

  • Aller sur http://127.0.0.1:8080/sarah/fitbit?start=true pour lancer le serveur.
  • Aller sur http://127.0.0.1:8080/fitbit/start pour enregistrer le FitBit

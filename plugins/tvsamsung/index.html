Le module TV Samsung permet d'envoyer des messages sur la TV.

Une requête HTTP SOAP est envoyé inspirée de ce qui est possible

Configuration

Indiquer l'adresse IP de la TV dans la configuration du plugin. La TV doit être visible sur le réseau. Penser à lui affecter une IP Fixe au niveau de votre routeur internet.

Paramètres

SMS
  • sms: le message à présenter
Schedule Reminder
  • subject: le sujet
  • reminder: le message de rappel
  • date1: la date de début
  • date2: la date de fin
Incoming Call
  • callee: Appelé
  • caller: Appelant

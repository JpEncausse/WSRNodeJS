Le module Gesture ne fait que recevoir et décoder les commandes gestuelles envoyées par WSRMacro.

Le paramètre g contient le type de gesture:

  • g=1 => Poignet gauche > Epaule gauche
  • g=2 => Poignet gauche < Epaule gauche
  • g=3 => Poignet droit > Epaule droite
  • g=4 => Poignet droit < Epaule droite
  • g=5 => Poignets > Epaules
  • g=6 => Poignets < Epaules
  • g=7 => Swipe gauche
  • g=8 => Swipe droit
Il faut editer le fichier gesture.gesture pour ajouter d'autres mouvements.

Règles

L'idée est d'utiliser le moteur de règles pour associer des gestes à des actions:

  • Déclencher des sons
  • Piloter (gauche/droit/avancer) un robot (Lego MindStorm)
  • Commander les slides d'un navigateurs (WebSocket)
  • ...

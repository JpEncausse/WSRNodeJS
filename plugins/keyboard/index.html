Le module keyboard permet de simuler des évènements souris ou clavier en appelant le programme CLI /bin/Keyboard.exe ou WSRMacro

  • text: simule le text frappé au clavier
  • press: simule l'evenement keypress avec le code clavier
  • down: simule l'evenement keydown avec le code clavier
  • up: simule l'evenement keyup avec le code clavier
  • activate: activate process with given name to front
  • run: run process with given name to front
  • mod: use a key modifier like CTRL

Un cas d'usage est de simuler suivant/précédant sur des slides powerpoint ou musiques spotify en utilisant les gestures Kinect.

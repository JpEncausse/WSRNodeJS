Le plugin demo2 permet d'expliquer simplement le code d'un module pour S.A.R.A.H.

Le plugin est décrit dans le fichier demo2.prop :

{
  "modules": { 
    "demo2": {
      "description": "Demo de plugin type Module",
      "version": "0.1"
    }
  }
}

Les phrases magiques sont définies dans le fichier de grammaire XML demo2.xml :

  • Sarah, tu es cool
  • Sarah, tu es chouette
  • Sarah, tu es sympa

<grammar version="1.0" xml:lang="fr-FR" mode="voice" root="ruleDemo" xmlns="http://www.w3.org/2001/06/grammar" tag-format="semantics/1.0">
  <rule id="ruleDemo" scope="public">
    <example>Sarah tu es cool</example>
    <tag>out.action=new Object(); </tag>

    <item weight="2">Sarah</item>

    <one-of>
      <item weight="1.2">tu es cool</item>
      <item>tu es chouette</item>
      <item>tu es sympa</item>
    </one-of>

    <tag>out.action._attributes.uri="http://127.0.0.1:8080/sarah/demo2";</tag>
  </rule> 
</grammar>

Sur correspondance, le script demo2.js est ensuite exécuté.

Le script fait simplement un callback avec la réponse « merci ».

exports.action = function(data, callback, config) {
  callback({'tts': 'merci'});
}

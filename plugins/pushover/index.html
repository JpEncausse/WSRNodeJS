Le module pushover permet d'utiliser le service Pushover pour envoyer des notifications sur les platformes iOS et Android. C'est le service qui a été choisit par IFTTT pour envoyer des notifications aux smartphones.

Le paramètre push permet de véhiculer le message à envoyer. Le titre du message est définit dans les propriétés.

Pour configurer le modules vous devez:

  • Installer l'application iPhone ou Androïd
  • Créer un compte et application pushover
  • Renseigner dans SARAH l'identifiant du mobile et de l'application
